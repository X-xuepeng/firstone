1.查看版本
git --version

2.配置Git用户名字 和 邮箱
git config --global user.name 'xuepeng'
git config --global user.email '344355481@qq.com'

3.创建Git仓库 或 远程从仓库克隆
git init git clone

4.查询状态
git status

5.工作区 暂存区 版本区

6.放入到暂存区
将工作区的文件添加到暂存区（如果是新文件git就可以追踪文件状态变化）
git add * | -A | 文件名
git add [option] arg

7.移出暂存区
git rm --cached 文件名

8.将暂存区的文件 提交 成一个版本
git commit -m remarks('如文件备注提交第几次')

8.查询仓库里都提交了什么
git log
